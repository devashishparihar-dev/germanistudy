# GermaniStudy Project Rules

## 1. Professional UI/UX Standards
- Treat every frontend task as an opportunity to build a premium, world-class product. 
- Use modern aesthetic principles: flawless typography scaling, subtle glassmorphism, appropriate whitespace, and harmonious color theory.
- Reject mediocre, generic, or "MVP-style" designs. If asked to build a component, build it like it belongs in an award-winning application.
- Prioritize micro-interactions. Use Framer Motion or CSS transitions to ensure every hover, click, and state change feels tactile and responsive.
- Always respect the `isDarkMode` / `.dark` toggle system and ensure contrast ratios are accessible and visually striking in both themes.
